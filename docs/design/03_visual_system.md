# Visual System

## 基本姿勢

ビジュアルは「派手さ」で勝つのではなく、
`構造の強さ + 質感の差` で印象を作る。

つまり、色やエフェクトより先に、
タイポグラフィ、余白、配置の強さを優先する。

## カラー

### ベース

- 背景: 白に近い薄いグレー、またはわずかに色温度のある明色
- 文字: ほぼ黒
- 補助色: 青、シアン、淡いバイオレットのいずれかを薄く使う

### ルール

- アクセント色は 1.5 色くらいの感覚に留める
- グラデーションは主役ではなく、空気感づくりに使う
- 全セクションで色を変えすぎない

### 避けるべきこと

- ピンク、シアン、紫を全部強く出す
- 発光色を多用する
- 文字色まで装飾的にしすぎる

## タイポグラフィ

### 方向

- 見出しは強く、大きく、短く
- 本文は読みやすく、信頼感を保つ
- 英語はラベル程度に限定し、日本語が主役

### ルール

- h1 は圧倒的に強くする
- h2 はセクションの論点が一目で分かる長さにする
- 本文は 2 から 4 行で区切る
- ラベルは小文字情報として整理に使う

## 余白

### セクション余白

- モバイルでも窮屈にしない
- PC では大胆に空ける
- 情報が増えるほど、余白設計を厳密にする

### カード内余白

- 見た目のためでなく、読み順を作るために余白を使う
- タイトルと本文、本文と補助情報の間隔は一貫させる

## 素材感

### 推奨

- 半透明サーフェス
- ほんのりしたレイヤー感
- うるさくない影
- 薄いグリッドやノイズ

### 非推奨

- 重すぎるガラス表現
- ベタなネオン
- 3D 風の立体影を多用すること

## レイアウト

### 方針

- 完全な単調1カラムにしない
- ただし複雑にしすぎない
- 左右の緊張感を意図的に作る

### 推奨パターン

- 左に論点、右にカード群
- 大見出しと補足情報を分離
- 同じカードでも少しだけ高さや位置に差をつける

## ビジュアルの判断基準

次の3つを満たしていれば方向性は正しいです。

- スクリーンショット1枚でも印象が残る
- コピーを読まなくても、ただの制作会社には見えない
- 5分見ても疲れない
